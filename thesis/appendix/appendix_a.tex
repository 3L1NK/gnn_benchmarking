\section{Additional Diagnostics and Audit Material}
This appendix provides auxiliary material that supports traceability, reproducibility, and reviewer-facing auditability.

\subsection{Recommended Reading Order for Diagnostics}
For practical model review:
\begin{enumerate}
  \item deterministic decision ranking,
  \item full run-level metric table,
  \item IC-versus-Sharpe and frontier geometry,
  \item cumulative equity trajectories.
\end{enumerate}
This order prioritizes deployable risk-adjusted quality over visually attractive but potentially regime-fragile equity paths.

\subsection{Audit Notes}
The final pipeline includes automated checks for:
\begin{itemize}
  \item duplicate run keys in ledger data,
  \item NaN metrics in thesis-critical report columns,
  \item plot point-count consistency,
  \item bibliography placeholder metadata in main references.
\end{itemize}
Any failed check blocks report generation to prevent silent corruption of thesis artifacts.
